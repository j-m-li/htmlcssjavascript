Une page WEB est un fichier texte pur sauvegardé avec l'extension ``.html`` au lieu de ``.txt`` .

Voici un exemple des plus simples : 

```
<html>
<body>
<h1>Bonjour</h1>
<p>à tous!</p>
</body>
</html>
```
